# Audio level meter
### Description
A sound detector on an ESP32 with adjustable sensitivity. Once the system registers a sound above a specific level the LED ring lights up.
I made the system simulation in Wokwi. I started from a simple version and added more complex components with each iteration.  
This project is for HORIZONS EUROPA.  

### Why I made it  
I wanted to experiment with different kinds of input and makes something really interactive. I was listening to music and thought 
'man, I wish I had one of those speakers with LEDs that light up according to music'. In the end I didn't to exactly that but something quite
close.
